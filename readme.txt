To check the last login time and the login log respective of users.

== Description ==
This plugin help you to check the Login Log and Last login time for the users. We can analyze how many days since last login for the respective users. No need to configure, simply activate When Last and you’re ready to go! Adds a custom column to your Users list of “Last Login” and a Login Log linked to that user. Also add a custom column "Days since last login" to count the user login days. 


== Frequently Asked Questions ==
= 1) Installation Instructions =
    1)  Upload the plugin files to the ‘/wp-content/plugins’ directory, or install the plugin through the WordPress plugins screen directly.
    2)  Activate the plugin through the ‘Plugins’ screen in WordPress.
    3)   Navigate to 'Users/All Users’ and data will be showing under ‘	Last Login’, 'View Login Log', 'Days since last login'.

= 2) What does When Last Login do exactly? =
    When Last Login allows you to see when last users have logged into your WordPress website. This is great for big sites that require  user management.
